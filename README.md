# Sparrow Sound Desk

Black-and-white ambient sound workspace built with Astro and React.

## Run locally

```powershell
npm install
npm run dev
```

Build for production:

```powershell
npm run build
```

## Free release with GitHub + Vercel

1. Create a new GitHub repository.
2. Push this project to GitHub.
3. Sign in to [Vercel](https://vercel.com/).
4. Click `Add New...` -> `Project`.
5. Import your GitHub repository.
6. Keep the detected Astro settings.
7. Click `Deploy`.

Vercel will automatically rebuild the site on every push to your connected GitHub branches.

## Google indexing

After the site is live:

1. Open [Google Search Console](https://search.google.com/search-console/about).
2. Add your Vercel domain or your custom domain.
3. Submit your sitemap URL, usually `https://your-domain.com/sitemap-index.xml` or `https://your-domain.com/sitemap.xml`.
