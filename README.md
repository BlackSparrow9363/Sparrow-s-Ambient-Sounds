# Sparrow Sound Desk

Black-and-white ambient sound workspace built with Astro and React.


# Link 
sparrow-s-ambient-sounds.vercel.app 

## Run locally

```powershell
npm install
npm run dev
```

Build for production:

```powershell
npm run build
```
